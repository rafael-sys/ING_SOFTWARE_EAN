This document describes the structure of the Schema directory, and 

provides instructions for creating the Sample Schemas.

========================================================================

1.	Structure of the Schemas directory

The Schema directory contains the following files:
---folder customer_orders
-- archivo Readme_customer.txt


Siga las siguintes instrucciones:

1. En el Directorio de trabajo db_sample_project, que ya ha sido convertido en directorio git,  cree una rama/branch que se llama db-sample-schemas-rama_customer_orders
2. Cambiese a la rama db-sample-schemas-rama_customer_orders
3. Copie el contenido la carpeta db-sample-schemas-rama_customer_orders al Directorio de trabajo (db_sample_project)
4. Modifique cada uno de los archivos que se encuentran en la carpeta  db-sample-schemas-rama_customer_orders agregando una linea de comentario al inicio de
cada archivo agregando su nombre, correo y carrera a la cual pertenece
5. Modifique este archivo (Readmi_customer.txt ) y al inicio del  archivo agregue su nombre, correo.
6. Haga un commit con el siguiente mensaje:  db-sample-schemas-rama_customer_commit_inicial
7. Modifique este archivo (Readmi_customer.txt ) retirando las lineas de Copyright ( desde la 7 hasta la 28). Grabe el archivo.
8. Revise el estado del proyecto
9. Haga un commit con el siguiente mensaje: db-sample-schemas-rama_customer_orders_commit_Readme_customer_retiro_copyright
